## 使用模板

- 克隆模板
``` sh
git clone https://github.com/leejiawang/vuepress-template.git
```
- 安装 vuepresss
``` sh
yarn install & npm install -g vuepress
```

- 预览页面
``` sh
yarn run docs:dev & npm run docs:dev
```

- 生成静态页面
``` sh
yarn run docs:build & npm run docs:build
```
